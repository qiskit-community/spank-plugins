# Example Jupyter Python Notebooks

## Install python dependencies
---
```bash
python3.11 -m venv ~/ibm_qcsc_daa_sim
source ~/ibm_qcsc_daa_sim/bin/activate
pip install jupyter
```

## How to Run
---
```bash
jupyter notebook
```
